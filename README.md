# 📂 WebDAV 文件同步器 WebDAV-File-Synchronizer

> 为安卓用户设计，实现手机本地与 WebDAV 服务器间的无缝数据备份。

---

## ✨ 一、主要功能
- **多协议支持**：HTTP / HTTPS
- **双向文件管理**：浏览(仅限图片)、下载、重命名、删除云端文件
- **自动/手动上传**：支持指定目录静默同步
- **结构化同步**：完整保持本地目录层级
- **自定义路径**：本地文件夹可同步到云端任意位置
- **并发控制**：1-10 个任务并发（适合大量小文件）
- **后台保活**：前台服务 + 唤醒锁，息屏不中断
- **运行日志**：详细记录同步过程，方便排查

---

## 🚀 二、快速开始

### 1. 登录 WebDAV
1. 输入服务器地址（例: `https://dav.example.com/dav/` 或 `http://192.168.1.10:5005`）
2. 输入用户名和密码
3. 点击“连接”

### 2. 基本文件操作
- **浏览**：点击文件夹进入，左上角箭头返回
- **上传**：右下角 `+` 号选择文件
- **删除**：文件右侧垃圾桶图标

### 3. 配置自动同步（核心功能）
1. 点击顶部 **设置** 图标
2. **授予权限**：允许“访问所有文件”
3. **添加同步目录**：
    - 本地路径：`/storage/emulated/0/DCIM/Camera/`
    - 云端路径：`/手机相册/`
4. 打开“**启用自动上传**”开关

### 4. 手动触发同步
回到主页 → 点击顶部 **Sync** 图标 → 立即扫描上传。

## ⚙️ 三、进阶技巧
- **并发设置**：宽带好 → 5-10；弱网 → 1-3
- **日志排查**：点击 **日志** 图标查看上传/失败记录
- **HTTP 支持**：内网可直接用 http:// 连接

## ❓ 四、常见问题 (FAQ)

| 问题 | 解决方法 |
| :--- | :--- |
| **设置了自动同步，文件没立刻上传？** | 默认每 15 分钟检查一次。点顶部“同步”手动触发。 |
| **上传失败，报 401 或 404？** | 401 → 用户名/密码错；404 → 网址格式不对（需 http/https 开头） |
| **返回桌面后同步停止？** | 开启“忽略电池优化”，检查通知栏是否有常驻通知。 |
| **下载的文件去哪了？** | 一般在 `/Download/` 文件夹，高版本安卓路径可能不同。 |
